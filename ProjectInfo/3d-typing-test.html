3D Typing Test

Platform: WebGL

Date: June 2025

This is a simple 3D Typing Test made in Unity 3D. The user selects a paragraph and begins a one-minute typing challenge. Letters appear in 3D space and react visually as you type. The game gives a fun and slightly gamified way to practice typing speed.

Features:

  • Selectable paragraphs
  • 1-minute timed test
  • Floating 3D alphabet letters
  • WebGL browser play (no install)
